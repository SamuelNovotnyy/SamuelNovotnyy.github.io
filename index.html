function toggleDM(init) { var pageStyle = localStorage.getItem("pageMode"); if (!init) { if (pageStyle == "dark") { pageStyle = "light"; } else { pageStyle = "dark"; } } document.documentElement.classList.remove("dark"); if (pageStyle == "dark") { document.documentElement.classList.add("dark"); } localStorage.setItem("pageMode", pageStyle); resetCols(); }

Prehľad funkcionality kódov 1 a 2

Kód naľavo je bežné spisovanie farieb a iných globálnych premenných používaných v celom dokumente. Robí sa to v každom programovacom jazyku, vrátane CSS. Takisto umožňuje jednu dôležitú funkciu, a to prepínanie týchto globálnych premenných jednou JS-native funkciou.

Kód napravo je JavaScriptová funkcia nazvaná toggleDM. Zodpovedná za prepínanie temného režimu na webovej stránke. Pri jej vyvolaní overuje aktuálny režim uložený v localStorage pod kľúčom "pageMode". Ak parameter init nie je poskytnutý alebo je nastavený na false, prepína režim medzi "svetlým" a "tmavým" módom. Ak je aktuálny režim "tmavý", zmení ho na "svetlý" a naopak.

Táto funkcia pracuje s koreňovým prvkom HTML dokumentu (document.documentElement) tým, že prepína prítomnosť CSS triedy ".dark" pomocou classListu. Ak je pageStyle nastavený na "dark", pridá triedu "dark" k koreňovému prvku :root, čím povolí temný režim. Ak pageStyle nie je " dark" alebo je nastavený na "svetlý", odstráni triedu "dark", čím vráti štýl stránky na predvolený "svetlý" režim.

Okrem toho táto funkcia aktualizuje localStorage, aby uložila aktuálny režim, zabezpečujúc, že vybraný režim pretrvá aj po obnovení alebo znovuotvorení stránky. Nakoniec volá funkciu resetCols(), na vykonanie úprav týkajúcich sa farebnej schémy na základe zmeny režimu. Tento kód je spúšťaný tlačidlom na stránke na prepínanie medzi tmavým a svetlým režimom webovej stránky.

Udalostný poslucháč "mouseover"

Úryvok kódu 3 je cyklus navrhnutý na iteráciu cez prvky v zozname listLen2. Pre každý prvok v tomto zozname sa kód snaží pripojiť udalostné poslucháče pomocou metódy addEventListener. Metóda addEventListener slúži na počúvanie konkrétnych udalostí, v tomto prípade "mouseover" a "mouseout", na prvkoch v zozname listLen2.

V rámci cyklu sa používa “immediately invoked function expression“ (IIFE), aby sa vytvoril nový rozsah pre každú iteráciu. V tejto funkcii sa pokúša pripojiť "mouseover" udalostný poslucháč k aktuálnemu prvku (listLen2[i]). Keď sa "mouseover" udalosť vyskytne na akomkoľvek z týchto prvkov, spustí sa vykonanie funkcie setPropsMouseover a prenesie sa aktuálna hodnota i ako argument do setPropsMouseover. Podobne sa pokúša pripojiť "mouseout" udalostný poslucháč na spracovanie situácie, keď sa kurzor pohne mimo prvku, a spustí sa funkcia setPropsMouseout s aktuálnou hodnotou i ako argumentom.

for (let i = 0; i < listLen2.length; i++) { (function () { try { listLen2[i].addEventListener("mouseover", function () { setPropsMouseover(i); }); } catch (e) {} })(); (function () { try { listLen2[i].addEventListener("mouseout", function () { setPropsMouseout(i); }); } catch (e) {} })(); }

Pripájanie funkcionality k "mouseover"

Funkcie setPropsMouseover(i) a setPropsMouseout(i) boli navrhnuté na prejavenie správania "mouseover" a "mouseout" udalostných poslucháčov, ako bolo nastavené v predchádzajúcom kóde. Tieto funkcie upravujú vlastnosti prvkov súvisiacich s udalosťami myši, aby vytvorili vizuálne efekty a interaktivitu.

Funkcia setPropsMouseover(i) mení vlastnosti prvkov pri výskyte udalosti "mouseover". Prijíma index i ako argument, potom iteruje cez prvky z listLen2 a upravuje ich štýly na základe hodnoty i. Konkrétne mení pozadie, farbu textu, šírku a aplikuje CSS filter na prvky v listLen2. Snaží sa tiež upravovať vlastnosti prvkov v cListLen2, prípadne upravuje ich šírku, ak je to možné. Je však dôležité poznamenať, že táto funkcia používa bloky try-catch, na spracovanie prípadov, keď určité prvky nemusia existovať alebo vlastnosti nie je možné upraviť, čím sa predchádza výskytu chýb a prerušeniu kódu.

Na druhej strane funkcia setPropsMouseout(i) resetuje vlastnosti prvkov späť do ich pôvodného stavu pri výskyte udalosti "mouseout". Podobne ako setPropsMouseover(i), prijíma index i ako argument a iteruje cez prvky v listLen2 a prípadne aj v cListLen2. Vracia pozadie, farbu textu, šírku a CSS filter späť na ich pôvodné hodnoty a štýly, čím im podstatne vráti zmeny aplikované funkciou setPropsMouseover(i).

Obidve funkcie slúžia na dynamickú úpravu vzhľadu a štýlu prvkov súvisiacich s udalosťami myši, čím zlepšujú interaktivitu a vizuálnu spätnú väzbu, keď používateľ umiestni kurzor myši nad tieto prvky alebo ho od nich odtiahne na webovej stránke.
